# Project Overview

## What is Shop Bijclo PHP?

Shop Bijclo is a Laravel-based e-commerce web application built with PHP 8.2+. It provides a complete online shopping experience with product management, shopping cart, wishlist, order processing, and payment integration.

## Core Purpose

A full-featured e-commerce platform enabling:
- Product browsing and filtering
- Shopping cart with coupon support
- Wishlist management
- User account management
- Order placement and tracking
- Admin dashboard for management

## Technology Stack

- **Language**: PHP 8.2+
- **Framework**: Laravel 12.0
- **Frontend**: Blade templates, JavaScript, Tailwind CSS
- **Database**: MySQL (via Eloquent ORM)
- **Image Processing**: Intervention Image
- **Shopping Cart**: surfsidemedia/shoppingcart
- **Payment**: VNPay integration

## Key Features

- **Product Management**: Categories, brands, product variants, multi-image support
- **Shopping Cart**: Add, update, remove items, coupon application
- **Wishlist**: Save products for later, move to cart
- **User Accounts**: Profile management, address book, order history
- **Admin Dashboard**: Analytics, order tracking, content management
- **Payment Gateway**: VNPay payment integration

## Getting Started

1. Install dependencies: `composer install`
2. Copy environment: `cp .env.example .env`
3. Generate key: `php artisan key:generate`
4. Run migrations: `php artisan migrate`
5. Start server: `php artisan serve`

## Architecture Summary

- **Pattern**: MVC (Model-View-Controller)
- **Authentication**: Laravel UI with role-based access (admin middleware)
- **State Management**: Session-based cart, database-persisted wishlist
- **API Style**: Traditional Laravel blade rendering with form submissions
